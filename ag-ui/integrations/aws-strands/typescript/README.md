# Strands Integration (TypeScript)

TypeScript client for the Strands integration using OpenAI models.

## Usage

```typescript
import { AWSStrandsAgent } from "@ag-ui/aws-strands-integration";

const agent = new AWSStrandsAgent({ url: "http://localhost:8000" });
```

## Development

```bash
pnpm install
pnpm build
pnpm dev
```
